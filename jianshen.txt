你的情况是典型的“上交叉综合征”（表现为：头前引、圆肩/肩膀内扣、含胸、溜肩/斜方肌上束紧张）合并“肌肉量严重不足”。

作为你的线上教练，我会为你制定一个为期12周（约3个月）的阶段性计划。目标是：先矫正体态（解锁自然宽度），再填充肌肉（创造视觉宽度）。

核心训练哲学（为什么这么安排？）
避免“练歪了”：你现在的身体是“前侧紧，后侧弱”。如果直接去练胸肌、练推举，会拉紧前侧，让内扣更严重，加重溜肩。所以前4周禁止做任何大重量的推类动作（如俯卧撑、卧推）。

“降伏”斜方肌：你的斜方肌上束（脖子到肩膀的肉）因为紧张代偿而显得“溜”。我们需要通过拉伸让它放松，同时激活它本来的“老板”——中下斜方肌和菱形肌，把肩胛骨拉回正确位置。

主攻视觉焦点：当肩胛骨摆正后，集中火力攻击三角肌中束，这是肉眼可见变宽的发动机。

第一阶段：矫正与激活（第1-4周）
目标： 松解紧张肌肉，激活沉睡肌肉，建立肩胛骨的本体感觉。这4周可能肌肉围度变化不大，但体态改善会让你挺拔，为后续增肌打地基。

频率： 每周训练4天，形式为：拉伸+激活

工具： 弹力带（轻型）、瑜伽垫、按摩球（或网球）

每天必做（早晚各一次，每次10分钟）：
靠墙天使：

做法：背靠墙站立，脚跟离墙一脚距离。臀部、上背部、后脑勺贴墙。双臂举起呈“W”状，努力让手臂和手背贴着墙，上下滑动。感受肩胛骨在肋骨上滑动。

组数：3组 x 15次。

门口胸肌拉伸：

做法：双手扶住门框两侧，手臂略高于肩，身体向前倾，感受胸大肌的强烈拉伸感。

组数：保持30秒 x 3组。

隔天训练（如周一、三、五、日）：
YTW伸展（俯身）：

做法：趴在床上或瑜伽垫上，额头点地。双手举过头顶呈“Y”字（大拇指朝上），感受下背部发力抬起手臂；然后双臂向两侧打开呈“T”字；最后屈肘呈“W”字。全程用肩胛骨夹紧的力量带动手臂。

组数：每个字母做3组 x 12次。

面拉：

做法：如果没有龙门架，可将弹力带固定在门把手上。双手抓住弹力带，大臂抬平，双手向脸部拉近，并向外旋转（想象把绳子拉向眉心，并展示肱二头肌）。

组数：4组 x 15次。这个动作是改善圆肩内扣的黄金动作。

第二阶段：核心力量建立（第5-8周）
目标： 在保持正确体态的前提下，开始引入全身力量训练，特别是强化后链，同时开始轻微刺激肩部肌肉。

频率： 每周训练4次，分化训练（上肢/下肢）

训练A：下肢+核心
深蹲（自重或轻哑铃）：强化全身力量基础。3组 x 12次。

平板支撑：3组 x 45秒。

训练B：上肢（侧重拉和稳定）
弹力带划船：坐姿，双脚蹬住弹力带，双手向后拉，夹紧肩胛骨。4组 x 15次。

哑铃侧平举（极轻重量）：使用1-2kg哑铃或两瓶矿泉水。这是关键动作。身体微微前倾，像“倒水”一样向两侧抬起，抬到与肩齐平即可。注意不要耸肩，如果耸肩，说明重量太重或斜方肌又在代偿。4组 x 15次。

俯身飞鸟：针对三角肌后束（改善圆肩非常关键）。俯身，手持轻物，像小鸟扇翅膀一样向后上方抬起。4组 x 15次。

第三阶段：增肌轰炸期（第9-12周及以后）
目标： 在体态正确的前提下，正式进入肌肉肥大训练，让肩膀宽起来。

频率： 每周4次，推拉腿分化

重点来了：肩部训练日（每周安排1个专门练肩日，或者把肩部动作放在推类日后面）
哑铃坐姿推举：现在可以做了！因为前8周你已经学会了稳定肩胛。选择能做8-12次力竭的重量。4组。

哑铃侧平举（重中之重）：增加一点重量（比如3-5kg），但要保证动作质量。可以采用“借力法则”：用一点惯力甩起来，然后控制着慢慢下放。关键是离心收缩。5组 x 12次。

杠铃直立划船（宽握）：宽握距可以更针对三角肌中束。4组 x 10次。

面拉：继续做，这是为了巩固战果，防止体态反弹。4组 x 15次。

关于饮食的精确建议（这是你能否成功的关键）
180cm/60kg，属于极难长肉的“外胚型”身材。

热量目标：每天摄入至少2800-3000大卡（比你现在的饭量要多出很多）。

蛋白质：每天摄入至少120g蛋白质（大约是：3个鸡蛋 + 200g鸡胸肉 + 一勺蛋白粉 + 一碗牛奶）。

加餐：如果你胃口小，必须把食物打碎喝下去。比如香蕉+牛奶+燕麦+一勺花生酱，打成奶昔，这杯液体热量很高且容易下咽。

总结你的待办清单
今晚：去买一个弹力带和一对2.5kg的哑铃（可拆卸的更好）。

第一周：忘记所有“提水桶”之类的土办法。每天早晚靠墙站5分钟，做“靠墙天使”。

第一个月：忍住不做俯卧撑，专心做YTW和面拉，把体态正过来。

第二个月：开始做最轻的侧平举，找肩部发力的感觉。

第三个月：加重量轰炸侧平举，同时每天多吃一顿饭。

请记住，你现在的纤细和溜肩，其实是块璞玉。一旦体态矫正，肉长对地方，穿衣效果会非常惊艳。坚持3个月，你会看到镜子里的自己明显变挺拔、变宽。
